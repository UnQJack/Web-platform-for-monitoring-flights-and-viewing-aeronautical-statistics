# Web platform for monitoring flights and viewing aeronautical statistics (PHP, HTML, CSS, MySQL, JavaScript, Python):
<p>• Developed a full-stack web platform for flight monitoring, booking management, and aeronautical analytics.</p>
<p>• Implemented an administrator panel with full CRUD functionality for scheduled flights and bookings.</p>
<p>• Automated flight lifecycle management by updating flight statuses from Active to Completed according to scheduled flight times.</p>
images/bookings.png
images/flights.png
<p>• Built a live analytics dashboard using Chart.js, displaying KPIs such as active/completed flights, total revenue, ticket sales, and popular routes, with automatic updates when database data changes.</p>
images/dashboard.png
<p>• Developed live flight tracking on an interactive Leaflet.js map, displaying active flights based on scheduled departure and arrival times.</p>
images/tracking.png
<p>• Designed a telecommunications monitoring module for recording signal strength, latency, bandwidth, packet loss, and connection status.</p>
images/telecom.png
<p>• Implemented a radar monitoring module that visualizes aircraft on a Leaflet.js map and calculates distance, azimuth, and detection status using aircraft and airport radar station coordinates.</p>
images/radar.png
<p>• Developed a Random Forest machine learning model to predict flight delays, delay probability, and operational risk using flight and telecommunications data.
images/ai_predictions.png
<p>• Implemented a Neuronal Network Autoencoder-based anomaly detection model to identify abnormal flight, telemetry, speed, and altitude data.</p>
images/anomaly.png
<p>• Integrated a Llama 3.1 virtual assistant capable of answering questions about flights, airlines, offers, and application data.</p>
images/chatbot.png
<p>• Implemented real-time notifications for aircraft position updates, bookings, flight modifications, radar detections, telemetry events, email delivery, and automatic flight status changes.</p>
images/messages.png
<p>• Developed an automated alert system for flight delays, anomalies, cancellations, and other operational events.</p>
